# ЗЕМЛЯ

Продающий каталог земельных участков на карте России.

- Карта с ценниками в стиле Авито
- Карточка участка: характеристики, описание, спутниковые фото (Esri World Imagery)
- Цена на витрине — **кадастровая стоимость**
- Данные из таблицы портфеля (кадастровые номера)

## Сайт

https://pavelabramyan.github.io/zemlya/

## Локально

Откройте `index.html` через любой статический сервер из корня репозитория:

```bash
python3 -m http.server 8080
```

## Данные

`data/plots.json` — участки с координатами, описаниями и ссылками на спутниковые снимки.
